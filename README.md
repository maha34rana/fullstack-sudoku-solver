# Sudoku Backtracking Visualizer

An interactive full-stack Sudoku solving visualizer built using React, Node.js, and Express that demonstrates the Backtracking Algorithm step-by-step with real-time animations.

---

## Features

* Sudoku solving using Backtracking Algorithm
* Real-time solving visualization
* Green glowing placement animation
* Red glowing backtracking animation
* Pause / Resume solving
* Dynamic speed control slider
* Easy / Medium / Hard sample puzzles
* Modern responsive UI
* Interactive Sudoku board

---

## Tech Stack

### Frontend

* React
* Axios
* CSS

### Backend

* Node.js
* Express.js

* Backtracking Algorithm
* Recursion

## How to Run Locally

### Clone Repository

```bash id="p9m3wp"
git clone https://github.com/maha34rana/fullstack-sudoku-solver.git
```

---

### Backend Setup

```bash id="q5x7lc"
cd server
npm install
npm run dev
```

---

### Frontend Setup

```bash id="r8v4qa"
cd client
npm install
npm run dev
```
